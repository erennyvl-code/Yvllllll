# YVL

> Premium YouTube Music Android app built with Flutter

Features:
- Ultra-dark pure black theme
- Animated sky gradient theme
- Bubble-style synced lyrics
- Karaoke mode with word-level highlighting
- Pill-shaped mini player
- More rounded album art corners
- No login required — direct home access
- Performance optimized player
- GitHub Actions APK build workflow
